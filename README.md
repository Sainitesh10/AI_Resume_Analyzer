## 🚀 Project Overview

**AI Resume Analyzer** is a web application designed to help job applicants optimize their resumes. By uploading a resume, the system extracts key information, matches it against job descriptions, and leverages AI to provide an ATS (Applicant Tracking System) compatibility score along with personalized improvement suggestions.

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript (Custom UI/UX)
- **Backend:** Python (Flask/Django - *handled by collaborator*)
- **AI/ML:** Natural Language Processing (NLP) for resume parsing

## ✨ Features

- 📤 **Resume Upload:** Easy drag-and-drop resume uploading.
- 🎯 **ATS Scoring:** Get a percentage score indicating how well your resume matches a target job description.
- 💡 **Actionable Feedback:** Receive AI-generated suggestions on missing keywords and formatting improvements.
